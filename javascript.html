getting started with javascript

JavaScript is a versatile programming language primarily used for building interactive and dynamic websites. Its purpose is to enhance the functionality of web pages by providing a means to manipulate the Document Object Model (DOM), allowing developers to dynamically update and modify the content and style of web pages without requiring a page reload. Over time, JavaScript has evolved into a powerful language that is not only essential for front-end web development but also widely used on the server-side (Node.js) and for mobile app development.

It is an interpreted language and not compiled. JavaScript is relatively compact, yet very flexible. Developers have written a variety of tools on top of the core JavaScript language, unlocking a vast amount of functionality with minimum effort. These include:

  • Browser Application Programming Interfaces (APIs) built into web browsers, providing functionality such as dynamically creating HTML and setting CSS styles; collecting and manipulating a video stream from a user's webcam, or generating 3D graphics and audio samples.
  • Third-party APIs that allow developers to incorporate functionality in sites from other content providers, such as Twitter or Facebook.
  • Third-party frameworks and libraries that you can apply to HTML to accelerate the work of building sites and applications.
KYL - Know Your Language (Variables and Data Types in JavaScript)

In order to know more about the language it's great to take a look at the core features of the language.

Variables

These are the containers in which the users can store different types of data. We can create one by using the let keyword.

let newVariable;

Notice the ;? It is there to specify that a line has ended.

Javascript is case-sensitive. That means newvariable is not same as newVariable.
Once a variable is declared you can assign some value to it. 

newVariable = "javascript";

Variables can also be declared by var keyword.

var TimeOftheDay = 'morning';
let orange = 10;

There are many types of data that we can store in variables. Some of the basic data types are:

let greeting = "Hello"; 	// string 
let age = 25;			// integer
let isAdult = true;		// boolean
let person = null;		// null
let job; 			// undefined

operators

Operators are the symbols that represent particular operations on different values and variables. These operators make the language capable of performing certain actions such as arithmetic calculations, assignment, comparison, and logical operations. 

5 + 8;		//Addition
10 - 3;		//Subtraction
10 * 3;		//Multiplication
100 / 10; 	//Division 
let num = 10;	//Assignment
num === 5; 	//Strict equality
!(true); 	//NOT
num !== "red";  //not-equal

These operators helps in evaluating and checking different conditions when they are paired with conditional blocks

Conditional Statements

Conditional statements allow you to make decisions in your code. These code structures are used to check whether a statement is true or false. The code then behave accordingly.

let age = 18;

// if-else statement
if (age >= 18) {
console.log("You are an adult.");
} else {
console.log("You are a minor.");
}

Here the code checks the if() condition and compares age variable with provided integer. If the result is true the first code block runs else the second code block is executed. One can add multiple checks by chaining the if...else conditional.

Functions

Functions can be defined as reusable code blocks. It's possible to define a body of code as a function that executes when you call the function name in your code. Javascript allow us to store a set of commands/operations in a function so that we don't have to rewrite the code everytime.

// Function declaration
function greet(name) {
	console.log("Hello, " + name + "!");
}

// Function invocation
greet("Alice");

In the above code the keyword function is used to declare a function named greet() which requires a name. This is likely to be a string data type. The code inside the {} is what is executed when a function is called.

At the time of calling a function to run. we write the name of the declared function (case-sensitive) followed by the data required in (). This will invoke the function and run the code in it.

In this example notice console.log()? It is also an in-built function which takes in a data type and logs it to the web-browser's console. This is mostly used for debugging the code.

let greet = function() { console.log("hello") };

greet();

We can also assign annonymous functions to a variable and call them as we would normally do.

objects

Objects are collections of data types. These consist of key-value pairs in key:value order. Where key is a string (or symbol) and value can be any data type including other objects. 

let person = {
  name: "John",
  age: 30,
  isAdult: true,
  sayHello: function() {
    console.log("Hello!");
  }
};

It can be observed that the object person has multiple properties such as name and age. It also contains a sayHello function that can be called as a method of this object.

person.name; 	//OUTPUT: 'John'
person.name = 'Stan';	//updating object property
person['age'] 	//OUTPUT: 30

The object's properties can be accessed and used by the above shown methods. It is also possible to update the properties of the object by using assignment operator =. 

person.sayHello()

With objects we can invoke their methods by using the above syntax. Where the person is the variable name assigned to the object, sayHello() is the method declared in the object. These are joined by a dot(.)

Arrays

Arrays are type of objects. Instead of using key-value pair, array uses numerical index numbers to access it's elements. It can be considered as a list.

let fruits = ["apple", "banana", "orange"];

The above syntax is used to declare an array. Here the data (strings) is enclosed in [] and separated by ,. 

fruit[0]; 	//OUTPUT: "apple" 
fruit[1]; 	//OUTPUT: "orange"

In order to access the elements of an array, integers starting from 0 are used. Where 0 depicts the first element in the array.

Similar to object. Array's elements can also be updated using the following syntax :

fruit[0] = "mango" 	// now fruit = ["mango","banana","orange"]

Loops

In JavaScript, loops are used to execute a block of code repeatedly. There are several types of loops, each serving different purposes. Here are three commonly used loop structures:

For Loop

This loop is usually used when you know the number of iterations beforehand. It consists of an initialization statement, a condition, and an iteration statement.

for (let i = 0; i < 5; i++) {
  console.log(i);
}

In this example, the loop initializes i at 0, this initalised variable acts as a counter. This variable can be declared in the loop declaration or outside the loop. The loop executes the block of code as long as i is less than 5, and increments i by 1 after each iteration.

While Loop

The while loop is used when the number of iterations is not known beforehand. It repeats a block of code as long as a specified condition is true.

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

This example achieves the same result as the for loop but uses a while loop. The block of code is executed as long as the condition i < 5 is true.

For...Of Loop

This specific loop is used for iterating over iterable objects, such as arrays, strings, or other iterable structures.

let fruits = ['apple', 'banana', 'orange'];

for (let fruit of fruits) {
  console.log(fruit);
}

This loop goes over each element in the fruits array in sequence. One element of the array is accessed and then stored in the variable fruit declared in the loop initialisation, then after the loop runs the code it moves to the next element in the sequence.

Messing with HTML (DOM Manipulation)

element selection

In order to create dynamic content in a webpage, the elements of the HTML has to be accessed. DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a document as a tree of objects, where each object corresponds to a part of the document, such as elements, attributes, and text content.

The DOM is used when we access the document object in Javascript. This is the top level object that holds all the elements and their properties.

If we have to change the text inside an HTML <p> tag, we will have to make an instance of the tag in Javascript so that we can access it's properties.

let textTag = document.querySelector("p");

we have successfully created an instance of the paragraph tag in Javascript and now we can use the object's property like innerHTMLto change the text in it.

textTag.innerHTML = "this is updated !";

There are many other ways in which we can access different elements of HTML file like:

  • getElementsByTagName:

    As the name suggest, this DOM selector gets all the elements that matches the given tag. document.getElementsByTagName("div"); returns a list that contains all the div.

  • getElementsByClassName:

    This gets all the elements with same class as passed as an argument to the method. document.getElementsByClassName('heading')

  • getElementById:

    In this, the elements are selected based on the Ids. This may look similar to the tagname and Class name but this sector only returns the first matched element while ignoring the remaining. document.getElementById('item')

  • querySelectorAll():

    This method returns all the elements that matches the specified CSS selector in the document. document.querySelectorAll('ul>li::after')
