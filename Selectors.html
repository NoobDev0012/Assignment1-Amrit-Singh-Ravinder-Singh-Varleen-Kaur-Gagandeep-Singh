getting started with the Selectors

In JavaScript, "selectors" typically refer to methods that allow you to select and manipulate elements in the Document Object Model (DOM).The DOM represents the structure of an HTML document, and JavaScript provides several methods for selecting elements within this structure.

Tag Selector

Selects all elements with a specific tag name.A "tag selector" in the context of CSS refers to selecting HTML elements based on their tag names. It allows you to style or target specific elements on a webpage by referencing their HTML tag. For example, if you want to apply a style to all '

' (paragraph) elements on a webpage, you would use the tag selector for the '

' tag. 

var elements = document.getElementsByTagName("tagName");
                /* CSS */
p {
    color: blue;
}

This selector is good if you want to style multiple elements of same tag.

ID Selector

Selects an element with a specific ID attribute.An "ID selector" in CSS is used to target a specific HTML element based on its unique identifier (ID).In HTML, the ID attribute provides a way to uniquely identify an element within a document. The ID selector in CSS allows you to apply styles or target specific behavior to that particular element

var element = document.getElementById("elementId"); 
                This is how it styled in /* CSS */
              #unique-id {
                background-color: yellow;
}

Attribute Selector

In JavaScript, there is no direct method called getElementByAttribute. However, you can achieve similar functionality by combining other methods such as querySelectorAll with attribute selectors. 

var elements = document.querySelectorAll('[attributeName="value"]');

Attribute Value Selector

Attribute selectors in CSS allow you to target elements based on the presence of specific attributes and their values. They provide a powerful way to style or manipulate elements in your HTML document.

var elements = document.querySelectorAll('[attributeName="value"]');

If you want to perform actions on each element matching the attribute criteria, you can loop through the NodeList obtained from querySelectorAll

var elements = document.querySelectorAll('[data-type="example"]');
                elements.forEach(function(element) {
                    // Do something with each element
                });
